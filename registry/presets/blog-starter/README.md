# @bloktastic/blog-starter

Preset with core content blocks for blogs and editorial landing pages.

## Included Packages
- `@bloktastic/hero`
- `@bloktastic/teaser`
- `@bloktastic/image`
- `@bloktastic/button`
- `@bloktastic/accordion`

## Install

```bash
bloktastic add @bloktastic/blog-starter
```
