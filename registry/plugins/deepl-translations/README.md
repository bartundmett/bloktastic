# @bloktastic/deepl-translations

Space plugin that integrates DeepL machine translation for automated and editor-assisted content localization across Storyblok languages.

## What It Does

DeepL Translations connects your Storyblok space to the DeepL API for high-quality machine translation:

- **One-click translation** of any story from its default language to target locales
- **Field-level control** — choose which fields to translate and which to skip (e.g., slugs, code snippets)
- **Rich text support** — preserves formatting, links, and inline elements during translation
- **Glossary integration** — use DeepL glossaries to enforce brand-specific terminology
- **Batch translation** — translate entire folders of content across all configured languages

## Use Cases

- Multi-market websites needing fast initial translations for editor review
- Blog content localization at scale
- E-commerce product descriptions across regions
- Reducing translation turnaround time from days to minutes

## Setup

1. Install the DeepL Translations space plugin in your Storyblok space.
2. Add your DeepL API key in the plugin settings (supports both Free and Pro tiers).
3. Configure your Storyblok space languages under **Settings > Internationalization**.
4. Open any story, select the target language, and use the translate action.

## API Key

You need a DeepL API key. Sign up at [deepl.com/pro](https://www.deepl.com/pro) for a Free or Pro plan. The Free tier includes 500,000 characters per month.

## Supported Languages

DeepL supports 30+ languages including English, German, French, Spanish, Portuguese, Italian, Dutch, Polish, Japanese, Chinese, and Korean. See the [DeepL docs](https://www.deepl.com/docs-api) for the full list.
